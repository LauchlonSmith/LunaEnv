# LunaEnv
Luna is a code environment based in python  

# Setup
1) Install python from 
   https://www.python.org/
2) Download Luna.py from releases 
# Usage
1) Copy luna.py to your project folder
2) File extension .l
# Example
main file (main.l)

luna env (luna.py)

open cmd

cd to project folder

type "luna.py"
